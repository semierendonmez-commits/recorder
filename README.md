## recorder (mod)

enhanced tape recorder. never miss a moment.

```
;install https://github.com/semierendonmez-commits/recorder
```

### usage

**K1+K2** — start/split recording. press once to start, press again to save current file and immediately start a new one (gapless split). recording indicator (blinking dot) appears in top-right of any script.

### features

- **smart naming**: `20260316_143022_cipher_170bpm_001.wav` — date, time, script name, BPM, sequence number
- **gapless split**: K1+K2 during recording saves and starts new file with zero gap
- **always-on mode**: enable in mod menu — recording starts automatically with every script
- **visual indicator**: small blinking dot top-right corner, toggle in mod menu
- **all files**: `dust/audio/recorder/`

### mod menu

E2 scroll, K3 action: toggle always-on, toggle indicator, manual start/stop/split. shows segment count and total recorded time.
